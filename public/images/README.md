# 360° Photos Directory

Place your Insta360 exported photos here.

## Insta360 Export Steps

1. **Open Insta360 Studio** and import your 360° photos
2. **Export** each photo as JPG (equirectangular format, 2:1 aspect ratio)
3. **Save** the exported files to this folder

## Usage

- Single photo: Use path `/images/your-photo.jpg` in `PanoramaViewer`
- Multi-scene tour: Add multiple photos and configure hotspots in `VirtualTour`

## File Format

- **Format**: JPG (equirectangular)
- **Aspect ratio**: 2:1 (e.g., 4096×2048 or 8192×4096)
- **Recommended**: Higher resolution for better quality (min 2048×1024)
